# Computing Resources

*Transcribed from: `Computing Resources.pdf`*

---

There are a number of computing resources available for use:

1. **Google Colab / Kaggle** -- These platforms provide access to GPU resources for running Jupyter notebooks, though sessions are limited to a few hours. It is recommended to develop and test your code in CPU mode first, then switch to GPU mode for final runs.

2. **Cloud Providers** -- Some cloud platforms, such as Microsoft Azure, offer free compute credits for students, which may be useful for training models.

3. **CS Department GPU Cluster** -- The School of Computer Science provides access to a GPU cluster managed via Slurm. To gain access, contact `system@cs.tau.ac.il`, stating that you are a student in this workshop (TA: Yoni Slutzky). A preliminary usage guide is available on Moodle. For additional storage resources, ask for the creation of a personal directory under `/home/yandex/DLWorkShop2026b`.
