## Project Proposal

PDF of up to 4 pages, specifying:
1. Team (up to three students)
2. End product (system from user's perspective)
3. Training and inference schemes
4. Datasets
5. Compute and storage requirements and options
6. Third party tools and models to be used
7. Existing solutions
8. Related work (papers, open source projects, etc.)

## Project Presentation

10 minutes for presenting slides (problem + solution) and demo.

## Project Submission

### Report
PDF of up to 8 pages, specifying all items covered by proposal (in greater detail), as well as:
- Lessons learned
- Benchmarking (where relevant)
- Potential extensions (future work)

### Code
GitHub repo (public or private). Code must include a demo and be executable "out of the box."
